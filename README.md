# Java Programming II MOOC - University of Helsinki

This repository contains my solutions to the exercises from **Java Programming II**, the second part of the Java MOOC provided by the University of Helsinki.  
This course builds on Java Programming I, diving deeper into **object-oriented programming, collections, streams, and GUI development**.

## 🛠️ Tech Stack & Tools

- **Language:** Java 11
- **IDE:** TMCBeans (NetBeans with the Test My Code plugin)
- **Testing:** TMC (Test My Code) framework for automated validation

## 📁 Project Structure

Each folder represents a specific **exercise from the MOOC course**.  
The internal structure follows the standard **Maven / NetBeans layout**:

src/main/java  
Contains the source code for the solution.

src/test/java  
Contains the automated unit tests provided by the university.

## 💻 How to Run the Exercises

1. Clone this repository:

git clone https://github.com/your-username/java-programming-ii.git

2. Open **TMCBeans**
3. Go to `File -> Open Project`
4. Navigate to the specific exercise folder
5. Click the **TMC "Test" button** to run the automated tests
